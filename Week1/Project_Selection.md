# Week 1: Project Selection and Foundation

## Selected Projects
The following projects have been selected based on their real-world application and alignment with industry-standard cloud patterns.

### 1. Project 1: Static Website with CDN 
- **Scope:** Global Content Delivery & Security
- **Tech Stack:** AWS S3, CloudFront, Route 53
- **Description:** Deploying a globally distributed static site. Focus is on leveraging edge locations for low-latency and securing the endpoint with SSL/TLS certificates.

### 2. Project 2: Serverless Function API
- **Scope:** Event-Driven Backend Logic
- **Tech Stack:** AWS Lambda, API Gateway, DynamoDB
- **Description:** Architecting a scalable REST API using a serverless approach. Focus is on "Cloud Native" compute patterns and NoSQL data modeling.
